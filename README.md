# Learn Piano Notes - Flutter

A Flutter app that quizzes you on Piano Notes

## Dependencies

- [sheet_music: ^0.3.0](https://pub.dev/packages/sheet_music)
- [url_launcher: ^5.4.5](https://pub.dev/packages/url_launcher)
